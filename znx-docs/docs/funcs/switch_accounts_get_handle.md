# switch_accounts_get_handle
!!! note
    This function is more often then not used with Extensions.

Gets a pointer for the account ID provided.

## Arguments
| Argument | Type | Description |
| ---- | ---- | ---- |
| Account ID | Int | ID of the user account |

## Syntax
```
switch_accounts_get_handle(id);
```

## Returns
The pointer of the account ID provided.
