# switch_accounts_network_service_avaliable
!!! warning
    This function requires connection to the Nintendo servers and/or a connected Nintendo account to the user, and is completely useless for homebrew.
Checks whether or not the provided user has a Nintendo account and/or has access to Nintendo Switch Online services.

## Arguments
| Argument | Type | Description |
| ---- | ---- | ---- |
| Account ID | Int | ID of the user account |

## Syntax
```
switch_accounts_network_service_avaliable(id);
```

## Returns
Whether or not the user can use online services.
