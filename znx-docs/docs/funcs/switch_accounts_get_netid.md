# switch_accounts_get_netid
!!! warning
    This function requires connection to the Nintendo servers and/or a connected Nintendo account to the user, and is completely useless for homebrew.
Gets the network ID for the account ID provided

## Arguments
| Argument | Type | Description |
| ---- | ---- | ---- |
| Account ID | Int | ID of the user account |

## Syntax
```
switch_accounts_get_netid(id);
```

## Returns
The network ID of the account ID provided.
