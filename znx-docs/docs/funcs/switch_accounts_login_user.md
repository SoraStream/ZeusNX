# switch_accounts_login_user
!!! warning
    This function requires connection to the Nintendo servers and/or a connected Nintendo account to the user, and is completely useless for homebrew.
Attempt to log the provided user into the Nintendo servers.

## Arguments
| Argument | Type | Description |
| ---- | ---- | ---- |
| Account ID | Int | ID of the user account |

## Syntax
```
switch_accounts_login_user(id);
```

## Returns
Whether or not the user was successfully logged.
