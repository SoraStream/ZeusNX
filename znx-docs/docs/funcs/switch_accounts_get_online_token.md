# switch_accounts_get_online_token
!!! warning
    This function requires connection to the Nintendo servers and/or a connected Nintendo account to the user, and is completely useless for homebrew.
Gets the Network Service Account Token for the account ID provided

## Arguments
| Argument | Type | Description |
| ---- | ---- | ---- |
| Account ID | Int | ID of the user account |

## Syntax
```
switch_accounts_get_online_token(id);
```

## Returns
The Network Service Account Token of the account ID provided.
