# ZeusBuilder
A simple enough tool to compile GameMaker: Studio 2 games for your Nintendo Switch!

## Currently this tool supports the current runtimes:
#### Monthly
    - 2023.8.2.152
    - 2023.11.1.160
    - 2024.14.3.260
    - 2024.14.4.268
#### LTS
    - 2022.0.3.99
Make sure to read the notes in the runtime downloader for any issues i've noticed!
## Prerequisites
    - GameMaker: Studio 2 (for the runtime)
    - a ZeusNX Runtime (Unpacked NSP from a VM GameMaker game targeting your version)
    - hacbrewpack
    - hptnacp
    - xdelta v3.1
