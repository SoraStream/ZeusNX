# ZeusBuilder
A simple enough tool to compile GameMaker: Studio 2 games for your Nintendo Switch!

## Currently this tool supports the current runtimes:
#### Monthly
    - 2023.8.2.152
    - 2023.11.1.160
    - 2024.14.3.260
    - 2024.14.4.268
#### LTS
    - 2022.0.3.99
Make sure to read the notes in the runtime downloader for any issues i've noticed!
## Prerequisites
    - GameMaker: Studio 2 (for the runtime)
    - a ZeusNX Runtime (Unpacked NSP from a VM GameMaker game targeting your version)
    - hacbrewpack
    - hptnacp
    - xdelta v3.1

## What's planned?
The general plan for now is to support every runtime from 2022.5 to latest, including LTS in the mix. I do want to support 2022.3 through 2.1.6, but i need to read up on how the asset compiler works for these runtimes.
Unfortunetly, runtimes 2.3.4.443, 2022.11.1.75 and 2024.6.1.208 will never be supported unless a retail switch game uses it, since yyg/opera removed the switch module from their servers.
